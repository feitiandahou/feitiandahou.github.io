## FocusTime 学习状态观察系统

一个功能完整的Web应用，用于记录和分析每日学习状态。支持所有参数的记录和数据持久化存储。

### 🎯 主要功能
- ✅ 完整的参数记录（Date, WakeUpTime, BedTime, FocusTime, ResultDescription, IsHoliday, IsSpecialEvent, SpecialEventDescription）
- ✅ 数据持久化存储（浏览器localStorage）
- ✅ 实时统计分析（今日、本周、本月数据）
- ✅ 历史记录管理（查看、筛选、排序）
- ✅ 数据导入/导出功能
- ✅ 现代化响应式界面设计

### 🚀 快速开始
1. 启动本地服务器：`python -m http.server 8000`
2. 访问：http://localhost:8000
3. 开始记录你的学习状态！

### 📊 参数说明

**parameters**

+ Date - 记录日期
+ WakeUpTime - 起床时间
+ BedTime - 睡觉时间  
+ FocusTime - 专注学习时间（小时）
+ ResultDescription - 学习结果描述
+ IsHoliday - 是否假期
+ IsSpicalEvent - 是否特殊事件
+ SpecialEventDescription - 特殊事件描述

详细使用说明请参考 [PROJECT_README.md](./PROJECT_README.md)
