# FocusTime 学习状态观察系统

一个基于Web的学习状态跟踪和观察系统，帮助用户记录和分析每日的学习专注时间、作息情况以及其他相关参数。

## 🎯 功能特性

### 📊 参数记录
根据README.md要求，系统支持记录以下参数：

- **Date** - 记录日期
- **WakeUpTime** - 起床时间  
- **BedTime** - 睡觉时间
- **FocusTime** - 专注学习时间（小时）
- **ResultDescription** - 学习结果描述
- **IsHoliday** - 是否假期
- **IsSpecialEvent** - 是否特殊事件
- **SpecialEventDescription** - 特殊事件描述

### 📈 数据统计
- 今日专注时间显示
- 本周平均专注时间计算
- 本月总专注时间统计
- 记录天数统计

### 💾 数据持久化
- 使用浏览器localStorage进行本地数据存储
- 支持数据导入/导出功能
- 数据格式为JSON，便于备份和迁移

### 🔍 历史记录管理
- 按日期降序显示历史记录
- 支持日期筛选功能
- 特殊标签显示（假期、特殊事件）
- 详细的时间线展示

## 🚀 快速开始

### 本地运行
1. 进入项目目录
2. 启动本地HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 打开浏览器访问：http://localhost:8000

### 使用说明

#### 1. 记录每日数据
- 打开应用后，系统会自动设置今天的日期
- 填写起床时间、睡觉时间、专注时间等信息
- 勾选假期或特殊事件（如适用）
- 点击"保存记录"按钮

#### 2. 查看统计数据
- 在"数据统计"区域查看各项指标
- 数据会自动更新，无需手动刷新

#### 3. 管理历史记录
- 在"历史记录"区域查看所有记录
- 使用日期筛选器查找特定日期的记录
- 点击"清除筛选"显示所有记录

#### 4. 数据导入/导出
在浏览器控制台中使用以下命令：
```javascript
// 导出数据
window.focusTimeApp.exportData();

// 导入数据（需要选择JSON文件）
// 暂无直接的UI导入功能，可以通过test.html页面测试
```

### 核心类和方法

#### FocusTimeTracker类
主要功能类，负责：
- 数据管理（增删改查）
- 统计计算
- 界面更新
- 事件处理

#### 关键方法
- `saveData()` - 保存数据到localStorage
- `loadData()` - 从localStorage加载数据
- `updateStats()` - 更新统计信息
- `renderHistory()` - 渲染历史记录
- `exportData()` - 导出数据为JSON文件

## 📊 数据分析建议

### 每日观察要点
1. **专注时间趋势** - 观察长期专注时间变化
2. **作息规律性** - 分析起床和睡觉时间的稳定性
3. **假期影响** - 对比假期和工作日的学习状态
4. **特殊事件影响** - 记录和分析特殊事件对学习效率的影响

### 长期主义实践
- 保持每日记录的习惯
- 定期回顾和分析数据趋势
- 根据数据调整学习计划
- 设置合理的专注时间目标
